# Окуйм - Платформа для обучения проект-менеджменту

## 🚀 О проекте

Окуйм - это интерактивная платформа для обучения управлению проектами и проект-менеджменту. Пользователи управляют виртуальными проектами, назначают задачи, распределяют ресурсы и реагируют на риски.

### 🎯 Особенности

- 📊 Симуляция управления проектами с реальными метриками
- 📈 Диаграмма Ганта для визуализации задач
- ⚠️ Система рисков с вероятностью наступления
- 📚 Теоретический материал с формулами и методиками
- 📉 Подробная статистика и аналитика
- 👑 Админ-панель для управления пользователями

### 🛠 Технологии

**Backend:**

- Node.js + Express
- Файловое хранение (JSON)

**Frontend:**

- React 19
- Tailwind CSS
- Chart.js для графиков
- Heroicons
